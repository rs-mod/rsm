admin is fat

## features

<details>
<summary><b>Dungeon</b></summary>

- **SS Solver**
    - Can support high skips
- **P3 Qol**
    - Titles and Sound for: Device completed, Section Completed.
    - Show Terminal Hitboxes
- **Terminal Solver**
    - A solver for terminals,
    - Queue
    - Supports Zero Ping, but you still need your own proxy
- **Pos Msg**
    - Positional Messages
    - Optionally can work for all players in the dungeon, can play sounds and show titles.
- **Puzzles**
    - Ice Fill
    - Quiz
    - Three Weirdos
    - Tic Tac Toe
    - Teleport Maze
- **Dungeon WP**
    - Waypoints for secrets and stuff
- **Abilities**
    - Ability keybinds for normal ability, and ult
- **Leap Rotate Fix**
    - Fixes leap rotation being incorrect sometimes
- **Mask/Immunity Status**
    - Tracks Spirit, Bonzo and Phoenix, renders duped masks as red, worn as gold
- **Leap Gui**
    - Custom Leap menu

</details>

<details>
<summary><b>Movement</b></summary>

- **Ether**
    - Etherwarp overlay
    - No Rotate
    - Fake Zero Ping Etherwarp/Zero Ping Teleport, (Does not actually move the player)
    - Zero Ping Interact
- **Null Binds/SOCD/Snappy Tappy**
    - Like wooting keyboard behaviour, if you're holding a and press d, a unpresses until you unpress d

</details>

<details>
<summary><b>Other</b></summary>

- **Sphinx Answer**
    - Answers the Mythological Ritual Sphinx question

</details>

<details>
<summary><b>Player</b></summary>

- **Chat**
    - Configurable message hiding with regex
    - Chat Emotes
- **Key Shortcuts**
    - Bind commands to keys
- **Chest Hit Fix**
    - Allows you to interact on chests while holding left click
- **Equipment Helper**
    - Middle click equipment menu
    - Instant close after click
- **No Place**
    - Stops unwanted placing of skulls & flowers with abilities
- **Protect Item**
    - Stops you from selling / dropping important or protected items
- **Slot Binding**
    - Allows for slots to be bound to hotbar
- **World Border Fix**
    - Stops world border preventing interactions which was not present on 1.8

</details>

<details>
<summary><b>Render</b></summary>

- **Hud**
    - Time Hud
- **Item Modifier**
    - Allows you to modifier item hover names
    - Allows you to change leather armour hex colour client side
- **OpSec**
    - Name Hider/Changer
    - Server ID Hider/Changer
    - Hypixel IP Hider/Changer
- **Visual Words**
    - Replaces text visually
- **Click Gui**
    - Options for customizing RSM's Click Gui
- **Crouch Speed**
    - Allows customisation of the crouch speed
- **Hide**
    - Hide Players (Range, P3, Device)
    - Hide Mini Wither
    - Hide dying
- **Image Hud**
    - Draws images on the screen
- **Jesus**
    - Jesus
- **Mana Star**
    - Shows Terraria inspired Mana stars for your Skyblock mana
- **Module List**
    - Renders a list of enabled modules
- **Screen Tint**
    - Tints the screen a configurable colour
- **Trail**
    - Renders a trail behind you
- **Waypoints**
    - Per island & dungeon room waypoints, separate to Dungeon Waypoints

</details>
